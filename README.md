# Storage, VMware & Backup Operations

This repository contains **read-only PowerShell scripts** focused on
operational visibility across **VMware, storage, and backup environments**.

The scripts are designed to support:
- Capacity monitoring
- Risk identification
- Backup verification
- Audit and operational readiness

They reflect real-world enterprise practices where changes are enforced
through approved processes, while engineers rely on scripts for visibility
and validation.

---

## VMware
Scripts focused on vSphere operational monitoring:
- Datastore capacity usage
- Snapshot risk identification
- VM-level visibility

---

## Storage
Scripts focused on SAN-backed storage visibility:
- Volume free space checks
- Disk performance indicators
- Capacity awareness for enterprise workloads

---

## Backup
Scripts focused on backup verification and audit readiness:
- Backup status validation
- Last successful backup checks
- Failure and warning detection

---

## Technologies
- VMware PowerCLI
- Windows Server
- SAN-backed storage (e.g. Pure Storage)
- Commvault (operational awareness)

---

## Disclaimer
Scripts are read-only and intended for monitoring and reporting purposes.
Always follow organisational change and backup policies in production.
